# Python 工具记录

这里记录刷题中学到的 Python 操作。目标不是背 API，而是知道什么时候该用它。

## 已学过或需要熟练的操作

| 操作 | 用法 | 常见场景 |
| --- | --- | --- |
| `enumerate(nums)` | 同时拿到下标和值 | 需要 index 和 value |
| `set(nums)` | 去重、快速判断存在 | 哈希去重、最长连续序列 |
| `dict` | 存 key-value | 计数、映射、索引表 |
| `Counter(s)` | 快速计数 | 字符频次、异位词 |
| `list.append(x)` | 末尾加入 | 栈、结果数组 |
| `list.pop()` | 取出并删除末尾 | 栈 |
| `list.pop(0)` | 取出并删除开头 | 队列模拟，注意较慢 |
| `"".join(chars)` | 字符列表转字符串 | 构造返回字符串 |
| `path.split("/")` | 按 `/` 切路径段 | 简化路径 |
| `del d[key]` | 删除字典 key | 计数变 0 后清理 |
| `all(cond for x in xs)` | 判断是否全部满足 | 覆盖、合法性检查 |
| `d.copy()` | 浅拷贝字典 | 保存一份不会跟着原 dict 改的快照 |
| `abs(x)` | 绝对值 | 距离、平方排序 |
